# ЦифроМилк (CifroMilk)

Веб-приложение для расчёта переработки молока в малых фермерских хозяйствах.

## Описание
Приложение позволяет ввести параметры партии цельного молока, выбрать направление переработки и получить расчёт оптимального использования компонентов (сливки, обрат). Все расчёты выполняются в браузере, данные сохраняются локально.

## Функционал
1. **Ввод данных:** Объём, жирность, плотность, белок.
2. **Сценарии:**
   - **Сливки, творог, молоко:** Расчёт нормализации питьевого молока, производства сметаны и творога.
   - **Сыр, сливки:** Расчёт нормализации для сыра (качотта), дозировок ингредиентов и выхода продукта.
3. **Журнал:** История расчётов с возможностью просмотра деталей и экспорта.
4. **Валидация:** Проверка данных, подсветка ошибок и высоких потерь (>5%).

## Инструкция по использованию
1. Откройте `index.html`.
2. Введите параметры молока и нажмите «Сохранить».
3. Выберите направление переработки.
4. Заполните параметры производства (потери, целевые жирности и т.д.).
5. Нажмите «Рассчитать».
6. Сохраните результат в журнал при необходимости.

## Формулы и коэффициенты
- **Сепарирование:** Баланс массы и жира с учётом потерь.
- **Нормализация:** Правило квадрата Пирсона.
- **Сыр:** Коэффициент К интерполируется (1.98 при 45% МДЖСВ, 2.16 при 50%).
- **Плотности:** Молоко (1.030), Сливки (1.010), Обрат (1.035).

## Технологии
- HTML5, CSS3, JavaScript (ES6+).
- LocalStorage для хранения данных.
- Адаптивный дизайн.

## Установка
Просто откройте `index.html` в любом современном браузере. Сервер не требуется.
